# インセプションデッキ テンプレート

## このドキュメントの目的

インセプションデッキは、アジャイル開発において**プロジェクトの方向性をチーム全員で共有する**ためのドキュメントです。
10の質問に答えることで、プロジェクトの本質を短時間で明確にできます。

### いつ使うか

- プロジェクト開始時にチームで認識合わせをしたい
- 短時間で本質的な議論をしたい
- ステークホルダーへの説明資料が必要
- スクラムやアジャイル開発を採用している

### 使い方

1. このテンプレートをコピーして、プロジェクト名に変更
2. チームまたはAIと対話しながら各質問に回答
3. 完成後、`PROJECT.md`と`MASTER.md`に関連情報を転記

### 関連ドキュメント

- [PLANNING_TEMPLATE.md](./PLANNING_TEMPLATE.md) - より詳細な企画書形式
- [USECASE_TEMPLATE.md](./USECASE_TEMPLATE.md) - 機能要件の詳細整理
