# 論理校閲コマンド

引数で指定されたファイル内の論理構造を検証し、矛盾や論理の飛躍を検出してください。

## 実行手順

1. `.claude/skills/proofread-logic.md` のスキル内容を読み込む
2. 指定されたファイルをスキャン
3. 主張・因果関係・条件文を抽出
4. 論理的な一貫性を検証
5. 結果をレポート

## 検証対象

- 同一文書内の矛盾する主張
- 因果関係の飛躍（A→C で B が省略）
- 暗黙の前提（「当然」「言うまでもなく」）
- 絶対表現（「常に」「必ず」「すべて」）
- 相関と因果の混同

## 引数

- `$ARGUMENTS` - 検証対象のファイルパスまたはディレクトリ

## 使用例

```
/proofread-logic books/ai-spec-driven-development-90percent/part1_why-ai-fails/
```
