# ファクトチェックコマンド

引数で指定されたファイル内の統計データ、研究論文への言及、事実に基づく主張を検証してください。

## 実行手順

1. `.claude/skills/proofread-facts.md` のスキル内容を読み込む
2. 指定されたファイルをスキャン
3. 検証が必要な箇所を特定し、ファクトチェックを実行
4. 結果をレポート

## 検証対象

- 数値・統計データ（「70%」「3倍」など）
- 研究論文・調査への言及
- 企業・製品に関する事実
- 歴史的事実・年号

## 厳密モード

すべての統計・主張に出典を要求します。出典がない場合は警告を出力。

## 引数

- `$ARGUMENTS` - 検証対象のファイルパス

## 使用例

```
/proofread-facts books/ai-small-is-accurate/part1_why-ai-fails/01_the-seventy-percent-problem.md
```
